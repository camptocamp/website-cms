===========
CMS Toolbar
===========

Render management actions for CMS contents.

Usage
-----

.. code:: xml

  <t t-raw="main_object.cms_render_toolbar()" />


Available actions/buttons
-------------------------

* create
* edit
* delete
* publish/unpublish

Also, there's a popover feature to remind users to publish contents.
